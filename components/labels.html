Labels Draft

Default label

Use sentence case only. Default alignment is right aligned. Long labels should wrap and should not be truncated.

<label class="tuiLabel">Label</label>

Required

Required fields should be marked with an asterisk (*). If all fields in the form are mandatory, you don't need to put asterisks everywhere, instead you should include information at the top of the form that states that all fields should be filled.

<label class="tuiLabel tuiLabel-is-required">Required label</label>
