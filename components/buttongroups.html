Button Groups

Use grouped buttons when there is a direct relationship between a given number of buttons. This pattern is normally used at the top of the page.

Default button

<div class="tuiButtonGroup">
   <button class="tuiButton">First button</button><button class="tuiButton">Mid button</button><button class="tuiButton">Last button</button>
</div>

</div>

Guidelines

Grouped buttons give users access to frequently performed, related actions. Only one button can be selected at a time, and initially, all buttons in the group are unselected.

  • Group buttons logically into sets based on their use and in order of importance
  • Ensure that button actions are clear and consistent
  • The most important action for a group set can be a primary button
