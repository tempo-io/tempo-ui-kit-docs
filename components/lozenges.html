Lozenges Draft

Text lozenges filled with colour, primarily used for displaying status. Available in both primary and subtle versions. Specific colors denote meanings that users can learn and recognise across our systems to become more efficient. Use primary lozenges by default. Consider using subtle lozenges for long tables with repeating lozenges, or when lozenges start to dominate the screen. Don't mix primary and subtle lozenges on a screen.

Lozenges Primary

Markup

Simply a span with base class and type class. When no type class added, the default lozenge gets a blue background color. (Design team?)

Default lozenge Blue Navy Blue Light Blue Medium Green Yellow Orange Red Pink Gray 
<span class="tuiLozenge">Default lozenge</span>
<span class="tuiLozenge tuiLozenge--blue-navy">Blue Navy</span>
<span class="tuiLozenge tuiLozenge--blue-light">Blue Light</span>
<span class="tuiLozenge tuiLozenge--blue-medium">Blue Medium</span>
<span class="tuiLozenge tuiLozenge--green">Green</span>
<span class="tuiLozenge tuiLozenge--yellow">Yellow</span>
<span class="tuiLozenge tuiLozenge--orange">Orange</span>
<span class="tuiLozenge tuiLozenge--red">Red</span>
<span class="tuiLozenge tuiLozenge--pink">Pink</span>
<span class="tuiLozenge tuiLozenge--gray">Gray</span>
